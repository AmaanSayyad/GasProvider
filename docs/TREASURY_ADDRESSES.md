This document lists all deployed Treasury contract addresses across supported chains for the Gas Provider Treasury Demo System. Each Treasury contract holds liquidity and executes real onchain distributions to users.

## Verification

### Verify Contract on Block Explorer

To verify the Treasury contract source code on block explorers:

```bash
cd contracts

# Verify on Coston2
npx hardhat verify --network coston2 0xc031c437d6b915dbdc946dbd8613a1ac9dd75d63

# Verify on Sepolia
npx hardhat verify --network sepolia 0x5b402676535a3ba75c851c14e1e249a4257d2265

# Verify on other chains (replace network name and address)
npx hardhat verify --network <network> <treasury-address>
```

## Notes

- All Treasury contracts use the same implementation (Treasury.sol)
- Each Treasury is independently owned by the deployer address
- Contracts are deployed on testnets for demonstration purposes
- For production deployment, use mainnet addresses and secure key management
- Treasury contracts hold real tokens and execute real blockchain transactions
